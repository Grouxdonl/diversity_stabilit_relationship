# diversity_stability_relationship
Dataset and R code

Dataset of bat, bird and butterfly communities and the R code to perform statistical analyses.

Variables descriptions :

- id : site id
- longitude : site longitude 
- latitude : site latitude
- region (only for bats) : site region (Ile de France or Manche, two French administrative regions)
- agriculture : level of agricultural intensification (PCA axis)
- urbanization : level of urbanization (PCA axis)
- species_richness : total species richness in the site
- weighted_mpd : weigthed Mean Phylogenetic Distance
- synchrony : index of population synchrony
- weighted_mean_cvpop : weighted mean population variability
- CV_com : coefficient of variation of community abundance
